-> Introduction
 
 This repository contains a collection of Image Processing Lab Experiments implemented using Python and OpenCV. The experiments are designed to help students understand basic to advanced image processing concepts such as image acquisition, enhancement, filtering, and transformation.

-> Objectives

1. To understand the fundamentals of digital image processing.

2. To learn how to capture, read, and manipulate images using Python.

3. To implement common image processing techniques.

4. To gain practical experience with OpenCV and NumPy libraries.

-> Tools & Technologies Used

1. Programming Language: Python 3.x

2. Library: OpenCV (cv2)

3. Supporting Library: NumPy

4. IDE: Visual Studio Code (VS Code)

5. Operating System: Windows
